## What's New
- **Standalone CLI binary** (`durak`) for Turkish NLP without Python dependencies
- **Core functionality:**
  - `durak tokenize` - Turkish-aware tokenization with apostrophe/clitic preservation
  - `durak normalize` - Unicode-aware normalization (İ/ı, I/i handling)
  - `durak version` - Build and version information
- **JSON output** (`--json` flag) for scripting and pipelines
- **Stdin support** for Unix-style text processing workflows

## Usage Examples

```bash
# Basic tokenization
durak tokenize "İstanbul'a gittik"

# JSON output
durak tokenize --json "Merhaba dünya!"

# Normalization
durak normalize "TÜRKÇE"

# Unix pipelines
cat corpus.txt | durak tokenize | sort | uniq -c
```
